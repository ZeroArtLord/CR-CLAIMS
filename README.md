# CR Claims Resolution - Premium Services Website

A modern, responsive website for CR Claims Resolution built with React, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Modern Stack**: React 18 + TypeScript + Vite
- **Responsive Design**: Fully responsive across all devices
- **UI Components**: Complete shadcn/ui component library
- **Performance**: Optimized with Vite for fast builds
- **Styling**: Tailwind CSS v4 with custom design system

## 📁 Project Structure

```
src/
├── app/
│   ├── App.tsx              # Main application component
│   └── components/          # Page components
│       ├── Header.tsx       # Navigation header
│       ├── HeroSection.tsx  # Hero section
│       ├── Footer.tsx       # Site footer
│       └── ui/              # Reusable UI components (shadcn/ui)
├── assets/                  # Static assets (images, logos)
└── styles/                  # Global styles
```

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## 🎨 Design System

This project uses a comprehensive design system with:
- **Color Palette**: Custom CSS variables for theming
- **Typography**: Responsive font scaling
- **Components**: 50+ shadcn/ui components
- **Animations**: Smooth transitions and micro-interactions

## 📦 Dependencies

### Core
- React 18.3.1
- TypeScript
- Vite 6.3.5
- Tailwind CSS v4

### UI Libraries
- shadcn/ui components
- Radix UI primitives
- Lucide React icons
- Emotion for styling

### Utilities
- React Hook Form for forms
- Recharts for data visualization
- date-fns for date handling
- clsx for conditional classes

## 🔧 Configuration

- `vite.config.ts` - Vite configuration with React plugin
- `tailwind.css` - Tailwind CSS configuration
- `tsconfig.json` - TypeScript configuration
